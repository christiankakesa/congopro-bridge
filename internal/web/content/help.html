Bienvenue dans le centre d'aide de Congopro Bridge. Retrouvez ici toutes les informations pour utiliser efficacement notre moteur de recherche d'entreprises.

Comment effectuer une recherche ?

Notre moteur de recherche intelligent vous permet de trouver des entreprises de plusieurs manières. Il vous suffit de taper vos mots-clés dans la barre de recherche :

  • Par nom : Tapez le nom exact ou partiel d'une entreprise (ex: "Guichet unique").
  • Par activité : Cherchez un domaine d'expertise ou un métier (ex: "Restaurant", "Banques", "Informatique").
  • Par ville : Ajoutez le nom d'une ville pour cibler géographiquement (ex: "Tourisme à Kinshasa", Assurances à Matadi).

Comprendre les résultats

Chaque résultat de recherche a été pensé pour vous donner accès aux informations essentielles en un seul clic :

  • Titre : Cliquez sur le nom de l'entreprise pour visiter directement son site web (si disponible).
  • Localisation : Cliquez sur l'adresse (accompagnée de la ville et du pays) pour ouvrir instantanément l'itinéraire sur Google Maps.
  • Contacts : Si un numéro de téléphone ou un e-mail est affiché, cliquez dessus pour appeler ou rédiger un message directement depuis votre appareil.

Astuces de recherche avancée

Si vous ne trouvez pas immédiatement ce que vous cherchez, voici quelques conseils :

  • Tolérance aux fautes : Notre moteur comprend les erreurs de frappe légères. N'hésitez pas à taper même si vous n'êtes pas sûr de l'orthographe exacte.
  • Mots-clés généraux : Si votre recherche est trop spécifique, essayez de retirer quelques mots.
  • Recherche sémantique : Congopro Bridge analyse le sens de vos mots. Vous pouvez chercher des concepts généraux et le système trouvera les entreprises dont la description correspond.
